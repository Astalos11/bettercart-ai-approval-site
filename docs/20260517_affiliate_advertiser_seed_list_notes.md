# Affiliate Advertiser Seed List Notes

Date: 2026-05-17

## Purpose

This is a starter list for post-network advertiser applications.

It reflects the updated strategy:

```text
apply broadly enough to measure approval rate
while still prioritizing food / grocery / nutrition / snack / beverage / supplement relevance
```

## Files

```text

```

## How To Use

After Awin / FlexOffers approval:

1. Apply to Tier A advertisers first.
2. Continue into Tier B and selected Tier C advertisers to measure approval rate.
3. Record every application and response in:

```text
docs/20260517_affiliate_application_tracker.csv
```

4. For approved advertisers, inspect:

- product feed availability
- deep-link support
- product URL
- image
- price
- SKU
- UPC
- GTIN
- MPN
- brand
- category
- terms around comparison / review / disparagement
